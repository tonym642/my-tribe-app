# Polls

## What It Is
Polls is a quick decision-making feature where advisors give short responses or vote on a question.

## Purpose
This feature helps you get fast, clear input without going into deep conversations.

## How It Works
- You ask a question.
- Advisors respond with short answers or votes.
- Results are displayed clearly.

## How To Use It
1. Enter your question.
2. Start the poll.
3. Review advisor responses.
4. Use the results to guide your decision.

## Key Rules
- Responses are short and direct.
- Designed for speed, not deep analysis.

## Example Use Cases
- “Should I take this deal?”
- “Is this a good idea?”
- “Which option is better?”

## Tips
- Use for quick clarity.
- Follow up with Advice or Debate for deeper insight.

## User Experience Goal
Feels like getting fast input from a trusted group.

## Future Improvements
- Weighted advisor voting
- User participation in polls
